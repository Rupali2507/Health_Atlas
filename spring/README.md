Technologies Used

    Spring Boot version 3.0.5
    
    Java17
    
    MySQL
    
    Build tool : Maven
    
Setup Instructions

Step-by-step guide to get the project running locally:

    Clone repository:
    git clone <repository-url>

    Navigate to project directory:
    cd project-directory

    Build the project:
    Using Maven: ./mvnw clean install

    Configure application properties:
    Update application.properties or application.yml with database credentials and other configurations.

    Run the application:
    Using Maven: ./mvnw spring-boot:run
